# 🛒 Ecommerce Frontend

Proyecto integrador individual del **Módulo 4 - Bootcamp Henry**.  
Aplicación de ecommerce desarrollada con **React + Next.js**, enfocada en la especialidad de **Frontend**.  

---

## 👨‍💻 Mi rol en el proyecto
- Desarrollo completo del **frontend**.  
- Implementación de componentes funcionales y vistas dinámicas (Navbar, catálogo, carrito, detalle de producto).  
- Integración con el backend provisto por Henry.
  

---

## 🚀 Tecnologías utilizadas
- **Frontend:** React, Next.js, TailwindCSS  
- **Backend:** API provista por Henry (Node.js, Express)
- **Base de datos:** PostgreSql  

---

## 📦 Funcionalidades principales
- Catálogo de productos dinámico (ejemplo: iPhone, MacBook, iPad, Apple Watch, AirPods, HomePod).  
- Carrito de compras con persistencia.  
- Autenticación de usuarios.  
- Vistas de detalle para cada producto.  
- Integración con API backend para gestión de productos y órdenes.
